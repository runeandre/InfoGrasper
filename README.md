## InfoGrasper
A really simple infoscreen cron solution for Raspberry Pi


## INSTALL HOW-TO:
1) Setup the Raspberry Pi by running the ../RaspberryPi_Setup/PiSetup file from command line
2) Setup the Windows content server by following the .../Infoskjerm/README.txt file

## Two components
1) Raspberry Pi to show images
2) Windows machine to convert PowerPoints and update the images on the Pi's
